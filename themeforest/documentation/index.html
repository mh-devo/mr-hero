Getting Started

Unzip the archive and find out the folder Mr.Hero. There are all the template files in this folder. You can open the following files for viewing in browser.

  • css folder with css files.
  • images folder with image files.
  • js folder with js files.
  • webfonts folder with font & fontAwesome files.
  • index.html Main Demo.

Fonts

2 fonts.. Felix-titling for titles and Questrial-Regular for description & paragraph

Felix-titling

Questrial-Regular

You will find the 2 fonts in webfonts folder

if you want replace them with another font..

  • Put your font in the webfonts folder
  • Open main.css file in CSS folder with your editor (for example, Brackets).
  • You will find font comment in the top, write font name in the font-family
  • write font location in src as picture below

Don't forget font type (for example, font-name.otf)

How to use?

  • Open main.css file in CSS folder and find out font comment, you will find it in the top.
  • Copy only the font-family line with font name
  • Find out body comment and past the line there as picture below
  • If you want specific font for specific paragraph,, past the font-family line in the paragraph as picture below

After you change the content and all is good Upload the template files (inside the folder Mr.Hero) to server and enjoy :)

How to change description size, color and height?

How to change background header image?

Open main.css file in CSS folder with your editor,, find out comment (Header image 1423 X 794) in header section

in background-image: url(../images/header-img.jpg) replace header-img with your image's name

in background-position show you specific region in the background image when resize (for example, john doe located in the center of the background image so we set background-position to 50% 50% so when resize window background fixed to center)
